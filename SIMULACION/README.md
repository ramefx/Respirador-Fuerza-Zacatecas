# Respirador-Fuerza-Zacatecas
 Respirador
 Programa utlizado en la simulación Proteus 8.4
	-La librería para simular arduino están en la carpeta simulino,
	 se pegan en C:\Program Files (x86)\Labcenter Electronics\Proteus 8 Professional\LIBRARY
	 o en ProgramData (carpeta oculta) dependiendo de su versión de proteus.

 Para poder simular, es necesario conocer la ruta donde se guarda el .hex, esto se puede
 hacer en arduino, Archivo, preferencias, Palomear "Mostrar salida detallada".
 Posteriormente se compila y se indicará la ruta del .hex, esta hay que ponerla en el 
 arduino "Doble click al arduino, pegar en program file:"
